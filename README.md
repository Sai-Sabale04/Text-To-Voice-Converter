# Text-To-Voice-Converter
Text-to-Speech Converter Web Application.
Developed a responsive and user-friendly Text-to-Speech Converter web application using HTML, CSS, and JavaScript. The application converts written text into natural-sounding speech using the browser’s Speech Synthesis API. It supports small, large, and complex text inputs and provides speech output in multiple languages and voices.
Key features include:
* Multi-language voice support.
* Clean and responsive user interface.
* Real-time text-to-audio conversion.
* Support for long paragraphs and complex text.
<br>
This project demonstrates skills in frontend web development, JavaScript APIs, UI design, and user experience development.
